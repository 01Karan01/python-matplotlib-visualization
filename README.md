# Matplotlib Data Visualization

This repository contains a Jupyter Notebook demonstrating data visualization
techniques using Python's Matplotlib library.

## Topics Covered
- Line plots
- Bar charts
- Scatter plots
- Histograms
- Subplots and figure customization

## Tools Used
- Python
- Matplotlib
- Jupyter Notebook
- Pandas, NumPy

## Purpose
To practice and demonstrate the use of Matplotlib for clear and meaningful
data visualization as part of a data analysis workflow.
